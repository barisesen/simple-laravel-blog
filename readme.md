# Simple Laravel Blog
Laravel is an example of a simple blog when I try to learn.

## Installation
 - Install laravel environment
 - clone this repo
 - cp .env.exampple .env
 - update .env file
 - php artisan migrate
 - php artisan db:seed
 - php artisan serve
 - open localhost:8000
 
## Features
  - Add new post, update, destroy
  - Check user authorizations
  - Check user is admin?
  - Add new comment
  - Simple form validation
